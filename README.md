# Neumorphism_Progress

# Installation :

Install using npm inside pages directory

```npm init -y```

```npm i hmos-neumorphism ```


Usage:
```html
<neuprogress progress="80%" width="300px" color="" height="20px" border="50px"></neuprogress>
```
